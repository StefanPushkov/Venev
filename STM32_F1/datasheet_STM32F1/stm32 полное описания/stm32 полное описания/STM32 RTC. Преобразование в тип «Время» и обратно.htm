STM32 RTC. Преобразование в тип «Время» и обратно

Всем хороши часы в STM32, но есть одна трудность: имеется счетный регистр и все. В нем идет подсчет секунд, а нам нужно текущие часы, минути, секунды.
Чтобы получить их напишем две функции, осуществляющие нужные нам преобразования.
Одна будет преобразовывать текущие время в формат пригодный для записи в счетный регистр (назовем ее TimeToRtc), а вторая будет выполнять обратное преобразование (соответственно, ее имя RtcToTime).


Для удобства работы с форматом времени создадим специальную структуру:

/*                      Структура для хранения параметров времени                            */
typedef struct 
{
  unsigned char  hour;     //часы	       
  unsigned char  min;	   //минуты    
  unsigned char  sec;	   //секунды    
} RTC_Time;

Напишем фукцию, преобразующую время в формат времени RTC (т.е. в секунды):

//********************************************************************************************
//function преобразует  "Время" в секунды (если 0->00:00:00)                                //
//argument указатель на струтуру, хранящую время для преобразования                         //
//result   время в формате "секунды"                                                        //
//********************************************************************************************
uint32_t TimeToRtc(RTC_Time *time )
{
  uint32_t result;
  result  = (uint32_t)time->hour * 3600; //учесть часы
  result += (uint32_t)time->min * 60;    //минуты
  result +=           time->sec;         //секунды
 
  return result;
}

//********************************************************************************************
//function преобразует секунды в формат "Время" (если 0->00:00:00)                          //
//argument значение счетчика, указатель на струтуру "Время"                                 //
//result   none                                                                             //
//********************************************************************************************
 void RtcToTime( uint32_t cnt, RTC_Time *time )
 {
   time->sec = cnt % 60;               //получить секунды
   cnt /= 60;                          //количество минут
   time->min = cnt % 60;               //получить минуты
   cnt /= 60;                          //количество дней
   time->hour = cnt % 24;              //получить часы
 }

Используя эти функции и фукции обращения к счетному регистру, которые описаны здесь, напишем пример работы с часами. Задача следующая: записать в RTC время 21 час, 15 минут, 10 секунд, а затем прочитать его обратно.

/*                                  Пример работы с RTC                                      */
 
RTC_Time Time;          //структура для работы с типом "время"
uint32_t tmp;           //вспомогательная переменная
 
//загрузить в структуру нужное время
Time.sec = 10;
Time.min = 15;
Time.hour= 21;
 
tmp = TimeToRtc(&Time); //преобразовать время в формат "понятный" для RTC
RTC_SetCounter(tmp);    //записать время в счетный регистр RTC
//  .....                 что-то делаем
tmp = RTC_GetCounter(); //прочитать время из RTC
RtcToTime(tmp,&Time);   //преобразовать в "человеческий" формат

Собственно, вся премудрость. Ничего сложного.
Точно так можно преобразовывать время для сигнального регистра.
